Termos e condições

O preco.co.mz é uma plataforma que permite aproximar o consumidor final as empresas bem como permitir a cada determinar o preço dos produtos aqui disponíveis. Ao usar este sistema os utilizadores com concordam com o termos e condições abaixo.

Empresas registadas

Os termos e condições abaixo se referem a todas empresas dispostas na página para visualização do consumidor final.

São empresas registadas, todas as empresas que pretendem aderir ao sistema preco.co.mz disponibilizando seus produtos e serviços com os respectivos preços para seus clientes.

Os descontos gerados pelo preço são determinados pelas empresas registadas.

Os dados inseridos na criação de conta servem para partilhar informação de início de secção, bem como partilhar experiências sobre do utilizador.

A violação dos termos e condições pode implicar no bloqueio temporário da conta até regularização de tal violação. Os preços indicados no sistema devem ser os mesmos na loja.

Consumidor final

Os termos e condições abaixo se referem a todo consumidor final com perfil criado no preco.co.mz para visualização do consumidor final.

Os descontos gerados para o consumidor final serão validados na compra seguinte.

Os dados inseridos na criação de conta servem para partilhar informação relativa a actividade do consumidor final, bem como receber e dar retorno sobre as experiências do utilizador.

Os dados inseridos na criação de conta estarão disponíveis para o preco.co.mz e para a empresa a qual fizer uma encomenda ou compra.

A criação de conta é totalmente grátis.

Os descontos só ficam validos se outros pessoas usarem do link que partilhar para finalizar compras.
